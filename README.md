# Solder by Anirudh
This is my cool PCB that I created while learning about PCB's in Solder by Hack Club

## What I used?
 - 220ohm Resistor x 2
 - 4.7k ohm resistor x1
 - LDR x1
 - 2N3904 x1
 - 0.1uF Capacitor x1
 - Push Button x 1
 - LED x2
 - CR2032 Battery x1
